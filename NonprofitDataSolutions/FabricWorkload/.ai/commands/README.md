# AI Commands

This folder contains reusable command guides for AI assistants working on the Nonprofit Data Solutions Fabric workload.

## Commands

- `item/create.md` - create a new workload item type.
- `plan/*.md` - plan features, PRDs, brainstorming notes, and implementation tasks.
- `project/current.md` - summarize the current project state.
- `workload/run.md` - run the workload locally with the development gateway and dev server.
- `workload/deploy.md` - build and deploy a workload frontend.
- `workload/publish.md` - prepare workload publication steps.
- `workload/update.md` - update workload metadata and configuration.
